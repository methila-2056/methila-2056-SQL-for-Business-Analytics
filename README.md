SQL for Business Analytics
Overview

This project demonstrates the application of SQL for analyzing telecom customer churn data. The objective is to extract business insights, calculate key performance metrics, and support data-driven decision-making through structured querying and aggregation.

Objectives

Write basic and advanced SQL queries

Perform data aggregation using SUM, AVG, COUNT, and GROUP BY

Analyze customer churn patterns

Generate key business KPIs

Tools & Technologies

SQL (SQLite)

Python

Pandas

Analysis Performed

Customer segmentation and churn filtering

Revenue and usage analysis

State-wise customer distribution

Service call impact on churn

Business KPI calculations (Churn Rate, Revenue, High-Value Customers)

Key Insights

Higher service call frequency is strongly associated with churn.

International plan subscribers show higher churn probability.

Revenue contribution varies significantly across states.

How to Run
python sql_analysis.py
