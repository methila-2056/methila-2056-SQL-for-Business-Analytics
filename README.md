SQL for Business Analytics
Project Overview

This project demonstrates the practical application of SQL for Business Analytics using a telecom customer churn dataset. The objective is to extract actionable business insights through structured querying, aggregation, and analytical reporting.

The analysis focuses on identifying churn behavior, revenue trends, customer segmentation, and performance metrics that support data-driven business decisions.

Business Objectives

Develop and execute basic and advanced SQL queries

Perform data aggregation using SUM, AVG, COUNT, and GROUP BY

Analyze churn behavior across customer segments

Generate key business performance indicators (KPIs)

Translate raw transactional data into meaningful insights

Dataset

Dataset: Churn-BigML (80% sample)

The dataset contains telecom customer information including:

Account details

Call usage metrics (Day, Evening, Night, International)

Service interaction records

International plan subscription

Churn status (Target Variable)
